# Netflix Content Exploratory Data Analysis

## Project Overview

This project analyzes Netflix's content library using Python to identify trends in genres, release years, ratings, countries, and content distribution.

## Objectives

- Clean and preprocess Netflix data
- Perform Exploratory Data Analysis (EDA)
- Visualize trends using interactive charts
- Analyze genres, ratings, duration, and countries

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Google Colab

## Key Insights

- Movies account for around 70% of Netflix content.
- Drama is the most common genre.
- Content production increased significantly after 2015.
- The United States and India contribute the largest number of titles.
